<h1 align="center">File Converter</h1>

<p align="center"><b>A fully local, offline CloudConvert replacement. Drop files. Select formats. Done.</b></p>

---

You know the drill: you just need to convert one file, and suddenly
you're staring at a $10/month subscription, a "free" trial asking for your
card, or some sketchy website that wants you to *upload your document or
photo to who-knows-where*. Yeah, no thanks.

File Converter is a small, fast Windows app that runs on your computer
and lets you convert files local-only. And the whole thing runs on
**your** computer — your files never go anywhere.

## What it does

- **Over 100+ formats supported** — convert between images, PDFs, audio,
  video, ebooks, spreadsheets, Word documents, presentations, and data files.
- **Save Web Pages to PDF** — paste any live URL to capture the page exactly
  as a browser would (desktop or mobile view), saving the entire web page
  as a clean PDF.
- **Merge PDFs** — select multiple PDF files in your queue and combine them
  into a single document with one click.
- **Smart format filtering** — drop any mix of files into the app; it
  automatically sniffs the format and populates the dropdown with only the
  conversions valid for that format.
- **Dynamic engine status** — see exactly which optional engines (like
  LibreOffice or Ghostscript) are active on your system. If missing, those
  targets are grayed out with a simple install hint rather than hidden.
- **Batch download & ZIPs** — download all completed conversions at once,
  choose a custom target directory, or export everything together in a
  single ZIP archive.
- **The shortcuts you'd expect** — drag and drop files anywhere on the
  window, clear finished rows quickly, and use Ctrl+S to download all
  completed results instantly.

## Is it really free?

Yep. **Free forever.** No subscription, no account, no watermark, no
"premium" button lurking in a corner. There's no catch — this exists
because converting a file shouldn't cost more than the document is worth.

## Your files stay yours

Everything happens locally on your PC. The app doesn't upload, sync,
collect, or phone home about your documents — ever. The only thing it
fetches from the internet is web pages (when you ask to convert a live
URL to PDF). Full details in [PRIVACY.md](PRIVACY.md).

## How to get it

- 🛒 **Microsoft Store** — coming soon! One-click install, automatic
  updates, zero warnings. This will be the easiest way.
- 💾 **Direct download** — grab `FileConverter.exe` from the
  [Releases page](../../releases) and just run it. No installer, no admin
  rights. Heads up: since the direct download isn't store-signed yet,
  Windows may show an "unrecognized app" screen the first time — click
  **More info → Run anyway**. That's Windows being cautious with new apps,
  not an actual problem.
