# Privacy Policy — File Converter

_Last updated: June 2026 · App owner: Majed Aljunaidy_

File Converter is a local application. It runs entirely on your computer.

## Your files

- Files you open or convert are processed **on your own machine** by a local
  server bound to `127.0.0.1` (your computer only — not reachable from
  the network).
- Documents, files, and conversions are **never uploaded anywhere** and are
  discarded when the application closes.
- Converted files are saved only where you choose to save them.

## Network access

The application accesses the internet for exactly one purpose: loading the
web page you explicitly request to convert to PDF. No local file contents or
user data are ever included in these requests. The app works 100% offline
for all file-to-file conversions.

## Data collection

- No analytics, no telemetry, no accounts, no tracking.
- No personal data is collected, stored, or shared.
